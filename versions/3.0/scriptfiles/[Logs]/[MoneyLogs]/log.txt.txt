Для установки на хост:
В файле server.cfg
Поменять строку: plugins streamer sscanf pawncmd mysql md5
На: plugins streamer.so sscanf.so pawncmd.so mysql_static.so md5.so
ПОДКЛЮЧЕНИЕ БАЗЫ ДАННЫХ -> skriptfile/mysql_settings.ini -> туда указываем данные от базы

Для Windows:
Для запуска на Windows поменять только подключения к базе данных.
Запускать фаил samp-server.exe
